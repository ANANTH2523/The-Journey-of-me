# Ananth.exe — The Journey

A humorous developer portfolio presented as if "Ananth" is a software product being reviewed before production deployment. Think GitHub meets Linear meets a futuristic OS boot screen.

**Live Demo:** Deploy with `npm run build` and serve the `dist/` folder.

## Tech Stack

- **React 19** + **TypeScript**
- **Tailwind CSS v4** (via `@tailwindcss/vite`)
- **Framer Motion** for animations
- **Vite 8** for bundling
- Dark mode cyberpunk aesthetic

## Sections

| Section | Description |
|---------|-------------|
| **Hero** | Terminal typing animation, gradient title, CTA buttons |
| **System Status** | Uptime, deployment location, core services dashboard |
| **Feature Modules** | Skills presented as installable software packages |
| **Bug Tracker** | Funny personality "bugs" styled as GitHub issues |
| **Changelog** | Life milestones as semantic version release notes |
| **Pull Request** | Career progress as a PR with checklist and reviewers |
| **Easter Eggs** | F1 card click, Konami code, floating developer memes |
| **Footer** | Test coverage bar (97.3%), subtle references |

## Easter Eggs

- **Click the Formula 1 card** → "Maximum speed mode activated"
- **Konami code** (↑↑↓↓←→←→BA) → "Developer mode unlocked"
- **Floating memes** appear randomly across the page

## Getting Started

```bash
npm install
npm run dev      # Start dev server at localhost:5173
npm run build    # Production build
npm run preview  # Preview production build
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Vite dev server with HMR |
| `npm run build` | TypeScript check + Vite production build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview the production build locally |
